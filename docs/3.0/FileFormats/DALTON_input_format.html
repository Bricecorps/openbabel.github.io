DALTON input format (dalmol)¶

Read Options¶

s Output single bonds only
b Disable bonding entirely

Write Options¶

a write input in atomic units instead of Angstrom
b write input using the ATOMBASIS format
k <basis>

specify basis set to use

e.g. -xk STO-3G
